.. _installation:

Standard use case is that you do not need to install this plugin manually,
but let Tessif_ handle it.

Manual Installation
*******************

Linux
-----

Install using a console with your virtual environment activated:

Latest Stable Version
^^^^^^^^^^^^^^^^^^^^^
.. code-block:: console

   $ pip install tessif-oemof-4-4

Latest Development Version (potentially unstable)
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block:: console

   $ pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple/ tessif-oemof-4-4

This installs the TestPyPI_ version of :code:`tessif-oemof-4-4` while resolving the dependencies on PyPI_.
